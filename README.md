## Installation

```bash
$ yarn install
```

## Running the app

```bash
# metro start
$ yarn start

# run android
$ yarn android
```

## android command guide

```bash
# clean android
$ yarn android:clean

# generate for test apk
$ yarn android:apk
```

## ENV

```bash
# development
# open cmd and type ipconfig (check IPv4 address of your computer)
# example:
API_URL="http://192.168.1.5:3001/"
```
